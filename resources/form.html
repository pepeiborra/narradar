The Narradar Termination Tool

LOPSTR'09 version

To start input a TRS in the textfield below in TPDB format, or a Logic program. Goals in logic program use the syntax 

%query: predicate(i,o,o)

Narradar recognizes a few extensions to the TPDB format

  • You can specify Narrowing programs using (STRATEGY NARROWING)
  • You can specify an Initial Goal using (STRATEGY GOAL <initial-goal>)
  • Note that you can use multiple STRATEGY sections
  • You can specify the pairs of a DP Problem using (PAIRS ..)
